### TACTICAL GLOBAL COORDINATE INFILTRATION INTERFACE

**GEO-TYPER** is a high-intensity, arcade-style typing challenge that transforms global geography into a tactical data-entry mission. 

---

## 🕹️ MISSION PROFILES

### [ 🎯 ] 20-CITY RACE
The standard competitive circuit. Navigate through 20 strategic global targets in the shortest time possible. 
*   **Parameters:** One typo results in an immediate mission failure.
*   **Goal:** Global record for the fastest successful infiltration sequence.

### [ ⚡ ] SURVIVAL MODE
An endurance-based challenge against a degrading system countdown.
*   **Parameters:** Every successful city-hack adds bonus time to your global clock.
*   **Reward Logic:** Perfect hacks (0 typos) grant 100% of remaining city-time; imperfect hacks grant 50%.
*   **Objective:** Maintain system uptime as long as humanly possible.

---

## 🖥️ INTERFACE & TELEMETRY

*   **Real-Time Map Telemetry:** Dynamic coordination using `Leaflet.js` to track your global flight paths.
*   **Tactical HUD:** A glass-morphic, reactive heads-up display that pulses with mission status (Standard, Warning, Danger).
*   **Mission Log:** Comprehensive post-mission debriefing with sortable metrics (Accuracy, Hack Time, Keystroke Sequence).

---

## 🚀 DEPLOYMENT

The system requires no backend infrastructure. Simply clone and execute `index.html`.

```bash
# Clone the repository
git clone https://github.com/mbleftley/Geo-Typer.git

# Navigate to the mission sector
cd Geo-Typer

# Initiate the interface
open index.html
```
